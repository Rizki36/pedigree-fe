# Pedigree Frontend

A part of [pedigree app](https://github.com/Rizki36/pedigree) that handles the frontend part

## Prerequisites

- Node.js ^20.19.1
- Yarn ^1.22.22

## Installation

- Clone this repository
- Run `yarn install`
- Run `yarn dev`

> [!IMPORTANT]
> If you change backend port, you need to adjust VITE_BASE_URL
